# Amrutam task

## Features:
- Responsive Design
- Card Scrolling with Dot Navigation
- dyanamic cards
- Minimalistic UI

## Technologies:
- React
- Tailwind CSS
- Vite

## How to Run Locally:
1. Clone the repository: `git clone <repo-link>`
2. Install dependencies: `npm install`
3. Start the app: `npm run dev`

## Notes:
- Ensure Node.js v16+ is installed.
